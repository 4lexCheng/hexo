安装nodejs
安装git
下载hexo

坑：一直下不下来

解决：更换镜像源

搭建仓库
  1. 仓库名：用户名.github.io
  2. 勾选public
  3. 勾选add a readme file
生成SSH Keys
  • 复制id_rsa.pub公钥
  • 打开github 用户/setting/SSH keys 粘贴保存
  • 测试(git bash)

回车，输入yes

本地生成博客内容

！！！文件夹名称不要带特殊字符，包括“-”

  1. 创建并打开文件夹
  2. 打开终端输入：

  1. 上线博客

    1. 编辑_config.yml文件
    1. 上传到github

    此时遇到坑

    解决方案

    1. 成功 hexo d以后，要在弹出的github认证窗口登录~
    2. 输入 github仓库名 链接访问
